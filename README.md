# ToDo4iOS
A todo list app for iOS and made by React Native.

## Components
```
NavigatorIOS
Modal
Realm
wolfy87-eventemitter
```

## Setup
1. **Clone the repo**

  ```
  $ git clone https://github.com/shiyuanhai/ToDo4iOS.git
  $ cd ToDo4iOS
  ```

2. **Install dependencies**

  ```
  $ npm install
  $ (cd ios; pod install)        # only for iOS version
  ```

3. **Running on iOS:**

  ```
  $ react-native run-ios
  ```
